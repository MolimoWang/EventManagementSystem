title Edit Account Permission - Basic Course of Events
participantspacing 2

actor "Administrator" as admin
control ": AccountSystem" as acs
participant ": AccountService" as asvc
entity "selected : Account" as sel

== 1. The administrator is on the 'View All Accounts' page (07-view_all_accounts.png) ==
== 2. The administrator selects the account to modify ==

admin -> acs: selectAccount(accountId)
activate acs

acs -> asvc: getAccount(accountId)
activate asvc
asvc --> acs: return account details
deactivate asvc

== 3. The system displays the account’s current permission level ==
acs -> admin: display current permission level

== 4. The administrator modifies the permission level ==
admin -> acs: changePermission(accountId, permissionLevel)
activate acs

acs -> asvc: updatePermission(accountId, permissionLevel)
activate asvc
note over asvc: The updatePermission method updates the account's permission\nin the database
asvc -> sel: setPermission(permissionLevel)
asvc --> acs: permission updated
deactivate asvc

acs -> acs: accounts := getAllAccounts()
acs -> acs: updateDisplay()

acs --> admin: 
deactivate acs

== 5. The system updates the view all accounts page with the updated account’s permission ==
