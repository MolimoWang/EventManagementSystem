title View All Accounts - 01 Basic Course of Events
participantspacing 5

actor "Administrator" as admin
control "<link:_>: AccountSystem" as acs
participant "<link:_>: AccountService" as asvc
entity "<link:_>a : Account" as acc

== 1. The administrator selects 'View All Accounts' ==
admin -> acs: viewAllAccounts(accountId)
activate acs

== 2. The system retrieves all existing accounts ==
acs -> asvc: getAllAccounts(accountId)
activate asvc

loop for every account
  asvc -> acc: readAccount(accountDetails)
end

asvc --> acs: return all accounts
deactivate asvc

== 3. The system displays a list of all accounts ==
acs -> admin: displayAccountList(accounts)

deactivate acs
