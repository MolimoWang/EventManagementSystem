== 2. The administrator attempts to assign an invalid permission level ==
admin -> acs: changePermission(accountEmail, permissionLevel)

note right of acs: The system checks if the selected permission level is valid
acs -> acs: validatePermission(accountEmail,permissionLevel)
activate acs
note right of acs: Validation fails because the selected permission\nis not among the allowed options

acs --> admin: display 'Invalid permission level'
note right of admin: The system prompts the administrator to select\na valid permission level
