title View Purchased Tickets Sequence Diagram

actor User
boundary TicketSystem
control TicketService
entity Ticket

note right of User: The user clicks 'Tickets' on their main page (34-view_all_events.png).
User ->> TicketSystem : viewPurchasedTickets()

activate TicketSystem
note right of TicketSystem: The system turns to the ticket list page (40-view_purchased_tickets.png) and displays all the tickets purchased by the user.
TicketSystem ->> TicketService : getPurchasedTickets(userEmail: String)

activate TicketService
note right of TicketService: The service retrieves the purchased tickets associated with the user's email.
TicketService ->> Ticket : readPurchasedTickets(userEmail: String)
activate Ticket
Ticket --> TicketService : return purchasedTicketsList
deactivate Ticket

purchasedTicketsList.size() > 0
    note right of TicketService: There are purchased tickets, proceed to display them.
    TicketService -->> TicketSystem : return purchasedTicketsList
    TicketSystem -->> User : displayPurchasedTickets(purchasedTicketsList)

deactivate TicketService
deactivate TicketSystem
