title List Upcoming Events Sequence Diagram

actor User
boundary EventSystem
control EventService
entity Event

note right of User: The user selects 'Upcoming' option from the main page (34-view_all_events.png).
User ->> EventSystem : listEvents()

activate EventSystem
note right of EventSystem: The system turns to list upcoming events page (36-list_upcoming_events.jpg) and displays the user's upcoming events.
EventSystem ->> EventService : getUpcomingEvents()

activate EventService
note right of EventService: The service retrieves the upcoming events from the database.
loop For each event in upcomingEventsList
    EventService ->> Event : getEventDetails(eventName)
    activate Event
    Event --> EventService : return eventDetails
    deactivate Event
end

upcomingEventsList.size() > 0
    note right of EventService: There are upcoming events, proceed to display them.
    EventService -->> EventSystem : return upcomingEventsList
    EventSystem -->> User : displayUpcomingEvents(upcomingEventsList)

deactivate EventService
deactivate EventSystem
