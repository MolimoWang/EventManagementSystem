title Administrator Modify Administrator Account

actor "Administrator" as a
control "AccountSystem" as b
participant "AccountService" as c
entity "all: Account" as d
entity "selectedAccount: Account" as e

==1. The administrator is on the view all accounts page **(07-view_all_accounts.png)**. 2.The administrator selects the account to edit ==

a -> b:showEditAccount(accountEmail)
activate b

b -> c:getAccount(accountEmail)
activate c

note right of c:The getAccount method is a standin for the future operation of retrieving account with the corresponding name.

loop for all accounts
  c -> d:getEmail()
end loop

c --> b: return selectedAccount
b -> b: displayEditAccountPage(selectedAccount)
note right of b:The system displays the modify account page **(13-edit_organizer_account.png)** which shows the details of the selected account as editable fields.

deactivate c
b --> a:
deactivate b

note right of a :The administrator can change two things of administrator: **Name** **Password**.

note right of a:The administrator enters the updated details of the account.
a -> b:updateAdministratorAccount(newName,newPassword).
activate b
note right of b:The administrator clicks "Submit" to confirm the modifications.

activate c

  c -> d:getEmail()
end loop

c --> b: return selectedAccount
deactivate c

activate c
