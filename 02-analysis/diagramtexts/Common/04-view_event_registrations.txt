@startuml
title View event registrations

actor "Admin/Organizer" as a
control ": TicketSystem" as s
participant ": TicketService" as u
entity "all : Tickets" as ae

note right of a: 1.The administrator or organizer wants to see the event registrations.

a->s:getRegistrations(eventName)
activate s

note right of s: 2.The system passes requests to the service.
s->u: getRegistrations(eventName)

activate u
activate ae

note right of u: 3.The service queries the registrations among all tickets.
loop for every ticket
  u->ae:getUser(eventName)
  ae -->u: return targetUser
end

deactivate ae

u-->s:return targetRegistrations

deactivate u

note right of s: 4.The system displays the registrations to admin/organizer .
s->s:displayRegistrations(targetRegistrations)

s-->a:
deactivate s

@enduml
